# Calculator App

## خطوات رفع المشروع على GitHub وتحميل APK

### 1. ارفع المشروع على GitHub
- اذهب على github.com وسجل دخول
- اضغط New Repository
- اسمه: calculator-app
- اضغط Create repository
- ارفع كل الملفات

### 2. GitHub Actions سيبدأ تلقائياً
- اذهب لتبويب Actions في المستودع
- انتظر 3-5 دقائق حتى ينتهي البناء

### 3. حمّل الـ APK
- بعد انتهاء البناء اضغط على الـ workflow
- ستجد Artifacts في الأسفل
- حمّل calculator-apk
